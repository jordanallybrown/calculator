# Calculator
A vanilla javascript calculator. 

You can view the calculator [here](https://jordanallybrown.github.io/calculator/). For iOS mobile devices, hide toolbar to see full calculator view. 

## Takeaways
- Evaluates the user's infix expression, converts to postfix, and then computes result. 
- Implement Stack (LIFO) using a Singly LinkedList as the underlying data structure
- Use the Factory Design Pattern to create different types of operators so that additional operators in the future are easily extended
